# Recruiter Notes

## One-Line Description

MedBot is an iPhone-ready React Native health assistant that performs preliminary COVID-19 symptom assessment and emergency warning detection.

## Resume Bullet

Developed a cross-platform health assistant with iPhone, Android, and web interfaces using React Native and Expo; implemented a rule-based symptom analysis engine for COVID-19 pattern detection, emergency warning escalation, and preliminary healthcare guidance.

## Demo Flow

1. Open the app in Expo Go.
2. Tap **Start assessment**.
3. Enter age and optional profile details.
4. Select symptoms such as fever, cough, or loss of taste/smell.
5. Review the risk score and next-step recommendations.
6. Try an emergency warning sign to see urgent escalation.
