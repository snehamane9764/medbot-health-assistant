# Project Overview

## Problem

People often need quick, understandable guidance when they are unsure whether symptoms may require testing, self-care, or urgent attention. MedBot demonstrates how a digital health assistant can collect structured symptom information and return practical next steps without claiming to diagnose a condition.

## Solution

MedBot provides a guided mobile assessment that asks for basic user details, current symptoms, temperature, and symptom duration. A deterministic rules engine scores the reported symptoms, identifies COVID-19-related patterns, and immediately escalates emergency warning signs.

## What Recruiters Should Notice

- Product thinking: the app includes onboarding, assessment flow, results, history, and safety disclaimers.
- Engineering clarity: the risk engine is separated from the UI and covered by automated tests.
- Mobile readiness: the project runs through Expo on iPhone and can be built with EAS.
- Healthcare safety: emergency symptoms are handled conservatively and clearly.

## Scope

This is a portfolio prototype and educational demonstration. It is not a medical device, diagnostic tool, or clinical decision support system.
